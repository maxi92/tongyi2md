# 通义转Markdown Chrome扩展

这是一个Chrome扩展，用于将通义千问的对话内容转换为Markdown格式。

## 功能

- 将通义千问对话转换为Markdown格式
- 支持一键转换
- 简单易用的界面

## 安装说明

1. 下载本项目
2. 打开Chrome浏览器，进入扩展管理页面（chrome://extensions/）
3. 开启开发者模式
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用方法

1. 在通义千问对话页面点击扩展图标
2. 点击"转换对话"按钮
3. 等待转换完成

## 开发说明

- manifest.json: 扩展配置文件
- popup/: 弹出窗口相关文件
- background/: 后台脚本
- content/: 内容脚本
- icons/: 扩展图标
